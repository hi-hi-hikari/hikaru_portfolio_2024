HIKARU　PORTFOLIO

  • ターゲット

    採用担当者

  • 目的

    転職活動にあたり、自身の経歴や強み、製作物を掲載することで、デザインスキルや思考を採用担当者に伝えるため。

  • 情報設計

    情報量が多い中でも、ご多忙な採用担当の方が必要な情報にすぐたどり着けること、十分な余白や図を使用して見やすくすることを意識しました。
    また、スキルだけではなく自分自身を知っていただくため、トンマナ設定ではレディライク/上品さ/洗練をキーワードに、ゴールドやセリフ体をアクセントに取り入れ、表現しました。ちょっとした遊び心も感じてもらいたく、全体的にシンプルながらも動きを取り入れました。

  • 製作範囲・期間

    企画・ワイヤーフレーム：７日
    デザイン：15日
    コーディング：30日

  • 使用ツール

    Illustrator／Photoshop ／ Visual Studio Code
